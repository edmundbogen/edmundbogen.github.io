GitHub + Claude Code

A two-part quick start for real estate professionals. Open a GitHub account, then connect Claude Code so you can save, version, and share your AI work like a pro.
The Edmund Bogen Team At Douglas Elliman Real Estate
Part 01 Start a GitHub Account
GitHub is where you'll store your prompts, skills, scripts, and AI-generated assets. Think of it as a digital filing cabinet that never loses anything and keeps a clean history of every change.
1
Go to GitHub
Open your browser and visit github.com. Click the Sign up button in the top right corner.
2
Enter Your Email
Use a professional email you'll keep long-term — your business address is ideal (e.g., name@yourdomain.com). Avoid throwaway accounts.
3
Create a Strong Password
15+ characters, stored in 1Password or your password manager. Enable two-factor authentication immediately after signup — non-negotiable.
4
Pick a Username
Choose something professional — your name or business handle. This becomes your public identity (e.g., github.com/yourname). Hard to change later, so pick well.
5
Verify Your Email
Open the verification email from GitHub and click the link. You're now in.
6
Create Your First Repository
Click the green New button on your dashboard. Name it something like my-ai-operations. Set it to Private. Check "Add a README file." Click Create repository. Done — you have your first repo.
Edmund's Rule
Default to Private repos for client work, business operations, and anything proprietary. Public is for open-source contributions and portfolio pieces only. Your prompts and workflows are your edge — protect them.
Part 02 Connect Claude Code
Claude Code is Anthropic's command-line tool that lets Claude work directly with your files and your GitHub repos. Once connected, you can ask Claude to build, edit, and commit code from your terminal — no copy-paste required.
1
Install Node.js
Claude Code runs on Node.js. Go to nodejs.org and download the LTS version. Install it with the default settings. One-time setup.
2
Open Your Terminal
On Mac: open Terminal (Cmd+Space, type "Terminal"). On Windows: open PowerShell or Windows Terminal. This is where you'll talk to Claude Code.
3
Install Claude Code
Paste this command and hit Enter:
npm install -g @anthropic-ai/claude-code
4
Install Git (if you don't have it)
Git is what connects your computer to GitHub. Mac users usually have it. To check, type git --version. If missing, download from git-scm.com.
5
Clone Your Repository
On GitHub, open your repo and click the green Code button. Copy the HTTPS URL. In your terminal, navigate to where you want the folder, then run:
git clone https://github.com/yourname/my-ai-operations.git cd my-ai-operations
6
Launch Claude Code
From inside your project folder, type:
claude
First time only: Claude Code will open your browser and ask you to log in with your Anthropic account. Approve it. You're connected.
7
Test It
At the Claude Code prompt, try:
# Ask Claude to build something > Create a simple buyer questionnaire template in markdown and save it as questionnaire.md
When you're happy with it, ask Claude to commit and push:
> Commit this and push to GitHub
What This Unlocks
You now have a versioned, AI-powered workspace. Every prompt, every skill, every script you build with Claude is saved, dated, and recoverable. Lose your laptop? Clone the repo on a new machine and you're back in business in 90 seconds.
